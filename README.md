# 🚀 JS & React Mastery Path: Panduan Belajar Mandiri

Selamat datang di repositori belajar JavaScript dan React! Repositori ini dirancang secara sistematis untuk membawa Anda dari tingkat pemula (nol) hingga mahir.

---

## 📖 Cara Mulai Latihan

Ikuti langkah-langkah di bawah ini untuk memulai perjalanan belajar Anda:

### 1. Pahami Peta Jalan (Roadmap)
Buka file [**`roadmap.md`**](roadmap.md). Di sana Anda akan melihat 4 Tahap utama pembelajaran. Fokuslah pada satu tahap dalam satu waktu. Jangan terburu-buru!

### 2. Pahami Struktur Folder
Buka file [**`struktur-folder.md`**](struktur-folder.md). Setiap folder di repositori ini sesuai dengan tahapan di roadmap. Gunakan folder ini untuk menyimpan seluruh kode latihan Anda.

### 3. Memulai Latihan (Contoh: Tahap 1)
Masuk ke folder latihan yang ingin Anda pelajari (misal: `01-javascript-fundamentals/01-logic-and-variables/`):
-   **Baca `explain.md`**: Pahami teori dasar materi tersebut.
-   **Ikuti `step-by-step.md`**: Tutorial praktis langkah demi langkah.
-   **Ngoding di `index.js`**: Tulis kode Anda sendiri di sini.

### 4. Menjalankan Kode di Browser
Hampir semua folder latihan dilengkapi dengan file `index.html`. 
-   Klik kanan file `index.html` > **Open with Browser** (Chrome/Edge).
-   Tekan **F12** pada keyboard untuk membuka **Developer Tools**.
-   Pilih tab **Console** untuk melihat hasil output dari `index.js` Anda.

### 5. Simpan Progres Anda
Jangan lupa untuk melakukan `git add`, `git commit`, dan `git push` secara berkala agar progres belajar Anda tersimpan dengan aman di GitHub.

---

## 🛠️ Tools yang Direkomendasikan
-   **VS Code**: Editor kode sejuta umat.
-   **Browser (Chrome/Edge)**: Untuk menjalankan dan men-debug kode.
-   **Node.js**: Opsional, untuk menjalankan JavaScript di luar browser.
