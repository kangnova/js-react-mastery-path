# 📘 Penjelasan: Logika & Variabel (JS Dasar)

Selamat datang di tahap pertama! Di sini kita akan mempelajari pondasi paling dasar dari JavaScript.

---

## 1. Variabel: Tempat Menyimpan Data
Di JavaScript modern (ES6+), kita menggunakan dua cara utama untuk membuat variabel:

-   **`const` (Constant)**: Gunakan ini untuk nilai yang **tidak akan berubah**. (Contoh: Nama Anda, Tanggal Lahir).
-   **`let`**: Gunakan ini untuk nilai yang **bisa berubah** di kemudian waktu. (Contoh: Skor game, Harga total belanja).

> [!TIP]
> Selalu gunakan `const` secara default. Hanya gunakan `let` jika Anda yakin nilainya perlu diubah.

---

## 2. Tipe Data Dasar
Beberapa tipe data yang paling sering digunakan:
-   **String**: Teks (misal: `"Budi"`, `'Halo Dunia'`).
-   **Number**: Angka (misal: `10`, `3.14`).
-   **Boolean**: Nilai benar atau salah (`true`, `false`).
-   **Array**: Kumpulan data (misal: `['Apel', 'Jeruk', 'Pisang']`).
-   **Object**: Data kompleks dengan label (misal: `{ nama: "Budi", umur: 20 }`).

---

## 3. Operator & Logika
-   **Perbandingan (`===`)**: Digunakan untuk mengecek apakah dua nilai sama persis (termasuk tipe datanya).
-   **Logika `&&` (AND)**: Benar jika **semua** kondisi benar.
-   **Logika `||` (OR)**: Benar jika **salah satu** kondisi benar.

---

## 🖥️ Cara Menjalankan di Browser
Untuk melihat hasil kode Anda, ikuti langkah ini:
1.  Buka file `index.html` menggunakan browser (Chrome/Edge/Firefox).
2.  Tekan tombol **F12** pada keyboard (atau Klik Kanan > Inspect).
3.  Pilih tab **Console**.
4.  Hasil dari `console.log()` dalam file `index.js` akan muncul di sana.
5.  **Setiap kali Anda mengubah kode**, jangan lupa **Simpan (Ctrl + S)** lalu **Refresh (F5)** browser Anda.
