# 🚀 JS & React Mastery Path: Dari Nol ke Mahir

Berikut adalah peta jalan (roadmap) dari nol hingga mahir, yang disusun secara sistematis agar Anda tidak merasa kewalahan.

---

## 🏗️ Tahap 1: Fondasi JavaScript (The Core)

Sebelum menyentuh React, Anda wajib menguasai JavaScript. React hanyalah JavaScript "biasa" yang diberi kekuatan super. **Jangan terburu-buru di fase ini.**

### 1. Dasar-Dasar Logika Pemrograman
- **Variabel & Tipe Data**: Pahami perbedaan `let`, `const`, dan tipe data seperti `String`, `Number`, `Boolean`, `Array`, dan `Object`.
- **Operator**: Penjumlahan, perbandingan (`===`), dan logika (`&&`, `||`).
- **Pengkondisian & Perulangan**: `if-else`, `switch`, serta `for` dan `while` loops.

### 2. DOM Manipulation (Interaksi dengan HTML)
Belajar bagaimana JS bisa mengubah teks di HTML atau mengganti warna tombol tanpa refresh halaman. Ini adalah cikal bakal cara kerja React.

### 3. Modern JavaScript (ES6+) — **WAJIB**
React sangat bergantung pada fitur-fitur baru ini:
- **Arrow Functions**: Cara singkat menulis fungsi.
- **Destructuring**: Mengambil data dari Array atau Object dengan cepat.
- **Spread/Rest Operator (...)**: Menyalin atau menggabungkan data.
- **Promises & Async/Await**: Cara mengambil data dari internet (API).

---

## 🎨 Tahap 2: Styling & UI Frameworks

Kapan harus belajar CSS, Bootstrap, atau Tailwind? Jawabannya: Sambil belajar JavaScript.

- **CSS Dasar**: Pelajari bersamaan dengan HTML sebelum masuk ke JS yang dalam. Pahami Box Model dan Flexbox.
- **Bootstrap**: Gunakan jika Anda ingin membuat tampilan cepat dengan komponen yang sudah jadi. Cocok untuk pemula yang ingin fokus ke logika dulu.
- **Tailwind CSS**: Sangat direkomendasikan saat Anda mulai masuk ke React. Tailwind sedang menjadi standar industri karena sangat fleksibel dan efisien.

---

## ⚛️ Tahap 3: Memasuki Dunia React.js

Setelah Anda nyaman dengan ES6, barulah masuk ke React.

### 1. Konsep Dasar React
- **JSX (JavaScript XML)**: Menulis HTML di dalam JavaScript.
- **Components**: Memecah website menjadi bagian-bagian kecil (misal: tombol, navbar, sidebar) agar bisa digunakan berulang kali.
- **Props**: Cara mengirim data dari satu komponen ke komponen lain (seperti memberikan instruksi pada robot).

### 2. State Management (Hooks)
- **useState**: Mengelola data yang bisa berubah (misal: angka counter, isi input form).
- **useEffect**: Menjalankan kode secara otomatis saat komponen muncul atau data berubah (misal: mengambil data dari database).

### 3. Routing & API Handling
- **React Router**: Membuat aplikasi "Multi-page" (berpindah halaman tanpa loading).
- **Axios/Fetch**: Menghubungkan React Anda dengan data nyata dari internet.

---

## 🏆 Tahap 4: Menuju Mahir (Advanced)

Di fase ini, Anda mulai membangun aplikasi yang kompleks dan profesional.

- **Context API / Redux**: Mengelola data global agar tidak pusing mengirim data lewat props yang terlalu dalam.
- **Form Management**: Menggunakan library seperti *React Hook Form* untuk formulir yang rumit.
- **Testing**: Belajar memastikan kode Anda tidak error menggunakan *Jest* atau *Cypress*.
