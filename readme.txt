🚀 JS & REACT MASTERY PATH: PANDUAN CARA MULAI LATIHAN
============================================================

Selamat datang di repositori belajar JavaScript dan React! 
Repositori ini akan membawa Anda dari tingkat pemula hingga mahir.

------------------------------------------------------------
Langkah-langkah Memulai:
------------------------------------------------------------

1. Buka File Roadmap (roadmap.md)
   Pahami 4 tahap utama yang akan Anda pelajari. Fokuslah pada satu 
   tahap setiap kalinya agar tidak kewalahan.

2. Pahami Struktur Folder (struktur-folder.md)
   Struktur ini dipetakan berdasarkan materi di roadmap. Gunakan 
   folder yang sudah disediakan untuk menyimpan kode latihan Anda.

3. Memulai Latihan (Contoh: Tahap 1)
   - Buka folder: `01-javascript-fundamentals/01-logic-and-variables/`
   - Baca: `explain.md` (Pahami teorinya)
   - Baca: `step-by-step.md` (Ikuti tutorialnya)
   - Tulis Kode di: `index.js` (Simpan dengan Ctrl + S)

4. Menjalankan Kode di Browser
   - Buka file `index.html` dengan Browser (Chrome/Edge).
   - Tekan F12 pada keyboard untuk membuka "Developer Tools".
   - Pilih tab "Console" untuk melihat hasil logikanya.

5. Sinkronisasi dengan GitHub
   Lakukan `git commit` dan `git push` secara rutin agar 
   progres belajar Anda tersimpan aman online.

------------------------------------------------------------
Tips Belajar:
- Konsistensi lebih penting daripada intensitas.
- Coba modifikasi kode sendiri untuk eksperimen!
------------------------------------------------------------

Selamat Belajar! ✨🏆
